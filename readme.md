# クリップボードに文字列をコピー

## 実行環境

* Windows

## 処理概要

* VBSからWSHを呼び出してクリップボードに文字列をコピーする

## 想定処理フロー

下記のように簡易的なID/パスワード入力ツールとして用いることを想定
1. クリップボードにIDをコピー
2. IDをペースト
3. OKボタン押下
4. クリップボードにパスワードをコピー
5. パスワードをペースト

## 文字コード

* 日本語を扱う場合は __Shift JIS__ で保存する

## MsgBox

* ここではメッセージを最前面に表示するように指定している
* 単純に表示するだけなら __MsgBox SendStr & "ID copied to clipboard."__ でよい
